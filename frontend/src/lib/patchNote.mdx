### 패치노트

#### v0.2.3

- 마작 플레이어 상세 페이지 추가
- 마작 페이지에서 우마 순위의 플레이어를 누르면 해당 플레이어의 상세 페이지를 띄워줍니다

#### v0.2.2

- 사이트 메인 페이지에 패치노트 표시
- 마작 페이지의 기록 카드에 카테고리 추가 표시 (ex: 3마 -> 3마 반장전)
- 기록 추가 페이지에 기타 기록사항 항목 추가
- 기타 기록사항이 추가된 경기는 최근 기록에서 초록색으로 표시됩니다

#### v0.2.1

- 통계 페이지 개선. 모바일 기기 등으로 보기 편하도록 너비 조정
- 사이트 기본 padding 조정
- 기록 추가 페이지에 카테고리 상세 선택 가능

### v0.2.0

- 마작 통계 페이지 추가
